# Portfolio 1: Synnestorm

Create a single page application (SPA) portfolio website that showcases your work. The portfolio must include the following projects: Cross Course Project, JavaScript 1 and Semester Project 1.

## Description

In my portfolio you can browse through all the projects I have developed during the last 2 years. Each project card has a link to the repository and live demo. I also have written a bit about myself, and my resume and cover letter is easily downloadable. The website is fully responsive across all devices.

## Built With

- HTML5
- CSS3
- JavaScript

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:synnestorm/portfolio.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
